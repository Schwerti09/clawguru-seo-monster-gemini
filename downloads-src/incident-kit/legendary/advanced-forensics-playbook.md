# Legendary: Advanced Forensics Playbook (High-Level)

## Purpose
High-level forensic workflow for complex incidents across hosts, cloud, and identity.

## Preserve
- Snapshots/disk images where possible
- Central log exports
- Chain-of-custody records

## Collect
- Process trees
- Network connections
- Persistence mechanisms
- Suspicious binaries + hashes

## Analyze
- Build a unified timeline
- Map IoCs across systems
- Identify patient zero

## Output
- Evidence inventory
- Timeline
- Root cause hypothesis
- Remediation roadmap
