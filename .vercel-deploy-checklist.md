# Vercel Production Deploy Checklist

## Vor dem Deploy: Environment Variables prüfen

### Sitemap-Konfiguration
- [ ] `SITEMAP_ALLOW_CHUNK=1` (aktiviert Runbook-Chunks)
- [ ] `SITEMAP_BUCKETS=5` (alle 5 Buckets: a-f, g-l, m-r, s-z, 0-9)
- [ ] `SITEMAP_RUNBOOKS_PER_BUCKET=2000` (max. Runbooks pro Bucket)
- [ ] `SITEMAP_100K_LOCALES=de,en` (Locales für 100k-Sitemaps)
- [ ] `PSEO_RUNBOOK_COUNT=1200` (Anzahl kuratierter Runbooks zur Laufzeit)

### AI-Provider (für Content-Generation)
- [ ] `AI_PREFERRED=<provider-order>` (Provider order from ENV)
- [ ] `GEMINI_API_KEY` gesetzt
- [ ] `OPENAI_API_KEY` gesetzt
- [ ] `DEEPSEEK_API_KEY` gesetzt

## Deploy-Prozess

1. **Vercel Dashboard öffnen**
   - Projekt: clawguru-seo-monster-gemini
   - Tab: Deployments

2. **Letztes Deployment auswählen**
   - Klick auf neuestes main-Deployment

3. **Redeploy mit Cache-Clear**
   - Button "Redeploy" klicken
   - **WICHTIG:** Haken bei "Use existing Build Cache" **ABWÄHLEN**
   - Redeploy starten

4. **Build-Logs beobachten**
   - Auf Fehler achten (npm warnings sind OK)
   - Build sollte in 3-5 Min durchlaufen

## Vercel Logs prüfen

Nach Deploy in Vercel → Functions → Logs filtern nach:
```
[SITEMAP] runbooks-
```

Erwartete Log-Zeilen:
```
[SITEMAP] runbooks-de-a-f: total=100, perBucket=2000
[SITEMAP] runbooks-de-a-f: filtered=XX
[SITEMAP] runbooks-de-a-f: top=XX, sample=aws-ssh,hetzner-docker,...
```

Falls `total=0`: Problem mit `materializedRunbooks()` oder `IS_BUILD_PHASE` Guard

## Troubleshooting

### Chunks immer noch leer?
- Vercel Logs prüfen: `[SITEMAP] runbooks-` zeigt `total=0`?
- Env-Variable `PSEO_RUNBOOK_COUNT` prüfen (sollte > 0 sein)
- `IS_BUILD_PHASE` könnte zur Laufzeit true sein (Bug)

### 100k-Sitemaps aktivieren (optional, für 400k+ Seiten)
In `app/sitemap.xml/route.ts` Zeile 58-59 auskommentieren:
```typescript
const pages100k = Math.ceil(count100kSlugs() / SITEMAP_PAGE_SIZE_100K)
const runbook100k = selectedLocales.flatMap((loc) => Array.from({ length: pages100k }, (_, page) => `${base}/runbook100k-${loc}-${page}.xml`))
```
Und in Zeile 72 hinzufügen: `...runbook100k,`
