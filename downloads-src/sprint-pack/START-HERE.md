# Welcome to ClawGuru

You just downloaded the **Incident Response & Hardening Sprint Kit**.

This pack is designed to help you run a focused security sprint:

- Hardening core services (SSH, Nginx, Docker)
- Setting up detection + basic response (Fail2ban)
- Running a repeatable security audit
- Using battle-tested incident response checklists and templates

## How to use this pack

1. Start with `runbooks/` to understand the procedures.
2. Use `checklists/` during the sprint to keep execution tight.
3. Run the scripts in `scripts/` to automate baseline checks.
4. Use `templates/` to document incidents and post-mortems.
