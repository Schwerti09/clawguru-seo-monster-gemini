# Legendary: Critical Infrastructure Defense (High-Level)

## Goal
Provide a blueprint for hardening and operating high-availability, safety-critical environments.

## Core themes
- Zero trust segmentation
- Immutable infrastructure
- Strong identity boundaries
- Offline recovery plans

## Controls
- Strict network segmentation (zones + conduits)
- MFA for all privileged access
- Break-glass accounts with offline storage
- Immutable backups + periodic restore drills
- Patch governance + emergency patch lanes

## Monitoring
- Baseline normal behavior
- Alert on configuration drift
- Continuous integrity checking

## Incident model
- Prioritize safety and availability
- Ensure manual fallback procedures
- Keep communications formal and logged
